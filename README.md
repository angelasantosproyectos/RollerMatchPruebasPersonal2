# 🛼 RollerMatch1

**La app de patinaje estilo Tinder** — Encuentra tu compañero de ruedas

---

## ✨ Características

- 🔐 **Registro & Login** con Firebase Authentication
- 🧭 **Onboarding** en 6 pasos para personalizar el perfil
- 🛼 **Swipe** estilo Tinder para descubrir patinadores
- ❤️ **Sistema de Likes y Matches** en tiempo real
- 💬 **Chat en tiempo real** con Firestore
- 🗺️ **Rutas** — Convocar y unirse a rutas de patinaje
- 🔍 **Búsqueda** de usuarios por nombre y @apodo
- 🌙 **Tema oscuro / claro**
- 📱 **PWA** — Instalable en móvil

---

## 🚀 Tecnologías

- HTML5 + CSS3 + JavaScript (Vanilla)
- Firebase Authentication
- Cloud Firestore
- Vercel (deploy)

---

## 📁 Estructura

```
RollerMatch1/
├── index.html          ← App principal (SPA)
├── estilos/
│   ├── main.css        ← Sistema de diseño global
│   └── app.css         ← Estilos de pantallas y componentes
├── js/
│   ├── app.js          ← Lógica completa de la app
│   └── firebase-config.js  ← Config Firebase (referencia)
├── manifest.json       ← PWA
├── vercel.json         ← Deploy Vercel
└── FIREBASE_SETUP.md   ← Guía de configuración Firebase
```

---

## ⚙️ Setup

1. Lee `FIREBASE_SETUP.md` para configurar Firebase
2. Pega tu config en `index.html` (busca `TU_API_KEY`)
3. Sube a GitHub y despliega en Vercel
